About SmartCourse

Project Overview

SmartCourse is an AI-powered course recommendation system designed to help students discover personalized learning paths. Using advanced Natural Language Processing (NLP) and machine learning algorithms, the system analyzes user queries and provides relevant course recommendations, ensuring a smart and efficient learning experience.

Dataset Information

The system uses a comprehensive dataset of 8,500+ courses collected from various sources. Each course entry includes:

  • Course Name
  • University / Provider
  • Difficulty Level
  • Course Rating
  • Course Description
  • Department / Subject Area

Preprocessing: The dataset has been cleaned to remove duplicates, missing entries, and inconsistencies. Text data has been normalized, lemmatized, and vectorized using TF-IDF and neural embeddings for accurate recommendations.

Technical Implementation

Backend

Flask-based REST API handles intelligent query processing. Supports:

  • POST /api/recommend
  • GET /api/history
  • POST /api/save
Real-time recommendations with TF-IDF and Neural models.

Machine Learning / NLP

- TF-IDF model for keyword-based course matching
- Neural Sentence-Transformer model for semantic understanding
- Cosine similarity scoring and ranking
- Model evaluation with precision@k, recall@k, hit-rate

SmartCourse ensures accurate, context-aware, and personalized course recommendations for every learner.
